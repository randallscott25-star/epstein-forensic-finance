Methodology & Data Sources

Wire transfers (directed edges with amounts): Extracted from Deutsche Bank SAR Exhibits A–E, authenticated via court exhibit numbers and bates stamps (DB-SDNY series). 481 wires, 228 entity-resolved counterparties, $558M total value across the master wire ledger. All amounts are (Unverified) automated extractions.

Entity co-occurrence (undirected edges with shared file counts): Computed from spaCy NLP entity extraction across 1,476,377 EFTA files. Two entities share a co-occurrence edge when they appear in the same document. Threshold: ≥20 shared files for display.

Banking relationships (dashed edges): Derived from entity co-occurrence filtered to financial document types. Indicates institutional-level document presence, not necessarily account relationships.

Node sizing: Proportional to total file count in the EFTA corpus. Node position reflects functional tier in the shell hierarchy as documented in the Master Wire Ledger (Phase 5I).

Shell entities not in wire ledger: Financial Trust Co., Epstein & Co., HBRK Associates, Outgoing Money Trust, Butterfly Trust, Insurance Trust, Epstein Interests, and Nautilus Inc. appear extensively in the EFTA corpus with financial document presence but have zero wires in the Deutsche Bank production. Their financial activity may be documented at other institutions.
